Assetto Corsa Competizione マルチプレイサーバ

このwebサイトはKunos Simulazioniによるレーシングシミュレータ「Assetto Corsa Competizione」のマルチプレイサーバ「Coppe ACC Server JAPAN」のポータルサイトです。Server List に「Coppe ACC Server JAPAN」という名前で登録されています。「Coppe」で検索を書けると見つかると思います。

このサーバはオンデマンドサーバです。管理者によってイベントが開催されたとき、または管理者の気分でサーバが稼働します。

次回のイベントはトップページの「upcoming event」カラムに表示されます (現在開発中です)。 また、「server status」カラムには現在（イベント開催状態や、管理者の気まぐれ等で)サーバが稼働しているかどうかが表示されます (現在開発中です)。

Mattemrost

Coppe ACC Server JAPANを使用してくださる皆様のコミュニティとして、mattermost (チャットサーバ、Alt Slackです)を運用しております。サーバに関する最新情報の掲載や、交流の場所として使用していく予定です。是非ご参加ください。

このページについて

このwebサイトは、認証されたユーザが次回のレースイベントを予約することができます。一般ユーザは、予約状況を事前に閲覧することができます。

予約されたイベントはAWS EC2インスタンスマネージャによってAWS EC2インスタンスを起動/終了されることで開催されます。

一連のシステムのソースコードはGitHubにて公開されています。皆様のContributionを歓迎しております。
